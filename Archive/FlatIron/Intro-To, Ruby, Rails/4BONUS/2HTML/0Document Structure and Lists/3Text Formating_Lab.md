##### `p`

##### Italic, Bold, Emphasis and Strong Tags

```html
<p>
  <em>Hello World!</em>
  <strong>Warning!</strong>
</p>
```

In **HTML**, we create lists using the `<ul>` tag, which stands for *unordered list*, along with the `<li>` tag for each *list item*.

```html
<ul>
  <li>Ham and Cheese</li>
  <li>Grilled Cheese</li>
  <li>Nacho Cheese French Fries</li>
  <li>Cheese soup</li>
  <li>Cheese and crackers</li>
  <li>Sushi</li>
</ul>
```
